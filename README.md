# Computer-Networks
Coursework from SCU covering Networks topics like TCP, UDP, pthreads.

CONTENTS:

Lab1: Explored functions calls versus system calls in C. In other words, open versus fopen.

Lab2: Copying multiple files concurrently using pthreads. In addition, calculating RTT time to open a web page.

Lab3: Created a client server file transfer program using TCP. tcpscon.c is a TCP server that can respond to multiple clients concurrently using pthreads.

Lab4: Created a client server program which sends data over a UDP connection to a server which copies the data to a file.

Lab5: Another UDP client server program, but made reliable. Uses checksum, sequence numbers, acks, and timer. The client simulated pack loss and handles retransmissions.

Lab6: Explored dijkstra's algorithm implementing a link state routing protocol.

Lab7 (NOT LISTED): Created a virtual network topology using GNS3. Explored basic functionaility using the console and played around with basic commands.
